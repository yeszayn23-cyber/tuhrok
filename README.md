# طُهر (Tuhr)
تطبيق "طُهر" هو رفيق رقمي ذكي للمرأة، مصمم كـ Telegram Mini App ليوفر تجربة أنيقة وآمنة.

## المميزات:
* **حساب الدورة:** تتبع ذكي مع تنبيهات.
* **الجانب الفقهي:** مكتبة شاملة لأحكام الحيض.
* **المساحة الترفيهية:** قسم خاص للراحة النفسية.

## التقنيات المستخدمة:
- HTML5, CSS3, JavaScript
- Telegram Mini App SDK
- Firebase (للتحقق وتخزين البيانات)
